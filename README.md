# dleen.github.io

Personal blog at https://blog.davidleen.me

## Development

```bash
# Preview locally
hugo server -D

# New post
./new-post.sh my-post-title

# Publish
git add -A && git commit -m "message" && git push
```

## Structure

- `content/posts/` - Blog posts
- `content/about.md` - About page
- `hugo.toml` - Site config
